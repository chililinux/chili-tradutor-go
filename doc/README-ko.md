# 칠리-번역가-고 🌶️

chilli-translator-go는 Go로 작성된 범용 기계 번역 래퍼입니다. 변수, 링크 및 기술 구문의 무결성을 유지하면서 스크립트(.sh, .py), 문서 파일(Markdown) 및 데이터 파일(JSON)을 번역하도록 설계되었습니다.

주요 장점은 Smart Cache v2.1.9로, 네트워크 호출을 대폭 줄이고 로컬 데이터 재사용을 통해 반복 번역 속도를 높입니다.

## ✨ 특징

* 다중 형식: .sh, .py, .md, .json, .yaml을 지원합니다.
* 구문 보존: 번역 프로세스 중에 셸 변수($VAR, ${VAR}), 마크다운 링크 및 문자열 자리 표시자를 자동으로 보호합니다.
* 병렬 번역: 고루틴(-j를 통해 조정 가능)을 사용하여 여러 언어를 동시에 처리합니다.
* 타임스탬프가 포함된 영구 캐시: 번역을 로컬에 저장하고 데이터 수명 주기를 관리하여 지능적인 정리가 가능합니다.
* 프로그레시브 인터페이스: 언어 코드 크기(예: en 대 zh-CN)에 관계없이 완벽한 시각적 정렬로 각 언어의 진행 상황을 실시간으로 표시합니다.

## 🚀 설치

Go가 설치되어 있고 시스템 종속성(gettext, trans)이 있는지 확인하세요.
```bash
자식 클론 https://github.com/chililinux/chili-tradutor-go.git
CD 칠리-번역기-go/src
go build -o chill-translator-go chill-translator-go-v2.1.9.go
sudo mv chill-translator-go /usr/local/bin/
```

파일을 표준 언어(EN, ES, IT, DE, FR, RU, ZH, JA, KO)로 번역하려면:

칠리-번역기-이동 -i meu_script.sh


### 언어 및 엔진 지정

cheli-treducer-go -and tutorial.md


### 캐시 삭제
지난 30일 동안 사용되지 않은 캐시 항목을 제거합니다.

칠리 번역기-go --clean-cache


## ⚙️ 옵션(플래그)

| 플래그 | 긴 | 설명 |
| :--- | :--- | :--- |
| -i | --입력파일 | 번역용 소스 파일입니다. |
| -e | --엔진 | 번역 엔진: Google, Bing, Yandex(기본값: Google) |
| -s | --소스 | 소스 언어(예: pt, en)(기본값: auto) |
| -l | --언어 | 쉼표 또는 모두로 구분된 언어 목록입니다. |
| -j | --작업 | 동시 번역 수(기본값: 8) |
| -f | --force | 로컬 캐시를 우회하여 강제로 변환합니다. |
| | --clean-캐시 | 오래된 캐시 항목(>30일)을 제거합니다. |
| -q | --조용함 | 자동 모드(시각적 진행 없음) |
| -v | --자세한 내용 | 실행하는 동안 기술 세부 정보를 표시합니다. |
| -V | --버전 | 현재 버전을 표시합니다. |

## 📁 출력 구조

* 스크립트/POT: ./pot/에 .po 파일을 생성하고 ./usr/share/locale/에 .mo 바이너리 파일을 생성합니다.
* 마크다운: ./doc/(예: README-en.md)에 번역된 버전을 생성합니다.
* JSON: ./translated/에 번역된 버전을 생성합니다.

## 🛡️ 캐시 로직(v2.1.9)

캐시는 ~/.cache/chili-tradutor-go/cache.json에 저장됩니다.

* 자동 마이그레이션: 이전 버전(v2.1.8)의 기록을 감지할 때 도구는 기록 데이터 손실을 방지하기 위해 레거시 기록에 현재 타임스탬프를 자동으로 스탬프 처리합니다.
* 자동 업데이트: 캐시에서 항목이 발견될 때마다 해당 항목의 "마지막 사용" 타임스탬프가 업데이트되어 향후 자동 정리로부터 항목을 보호합니다.
* 보안: --clean-cache를 통한 정리는 실제로 사용되지 않는 항목만 제거하여 번역 지식 기반이 건강하게 성장하도록 보장합니다.
